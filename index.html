Atividade com Git

Git é um sistema de controle de versão, desenvolvido por Linus Torvalds, onde é muito usado no desenvolvimento de Software, porém podendo ser aplicado à atividades diversas, como TCC's e Monografias. Este, na atualidade se tornou vital para trabalhar , seja em equipes ou individualmente, pois se tem um maior controle do que está sendo feito e principalmente em caso de erros é muito simples para voltar ao commit anterior, tirando a necessidade de alterar todo o arquivo, trazendo uma maior segurança no desenvolvimento do projeto.
